[[overview-http-status-codes]]
=== HTTP 상태 코드

|===
| 상태 코드 | 설명

| `200 OK`
| 요청 성공

| `201 Created`
| 리소스 생성 성공

| `204 No Content`
| 요청 성공 (응답 본문 없음)

| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 인증 필요

| `403 Forbidden`
| 권한 없음

| `404 Not Found`
| 리소스를 찾을 수 없음

| `500 Internal Server Error`
| 서버 오류
|===
